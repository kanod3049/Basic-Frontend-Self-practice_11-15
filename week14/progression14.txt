ในสัปดาร์นี้ผมได้ดูแลส่วน frontend ของ integrated project sprint 3 
ผมได้ทำการ Refactor เพื่อปรับปรุงโครงสร้างให้เป็นระเบียบตามคำแนะนำของอาจารย์ครับ
โดยแยกส่วนการเชื่อมต่อข้อมูล API ออกจากส่วนควบคุมการทำงานต่างๆ
และสร้างโมดูลสำหรับจัดการ Dialog/Modal แยกต่างหากตามคำแนะนำของอาจารย์ เพื่อลดความซ้ำซ้อนของโค้ด
และทำให้การจัดการ DOM และ Asynchronous Flow มีประสิทธิภาพมากขึ้นครับ
