ในสัปดาร์นี้ผมได้ดูแลส่วน frontend ของ integrated project sprint 4 
ผมได้นำหลักการ async มาใช้ร่วมกับ mock API เพื่อจำลองการทำงานของระบบจองรายวิชา 
ทำให้สามารถพัฒนา UI และ Logic การคำนวณหน่วยกิต ได้ล่วงหน้า นอกจากนี้ยังได้เพิ่มการตรวจสอบ Reservation Period 
ทั้งฝั่ง Client-side และการดักจับ Error Response จาก Server เพื่อให้ UX ราบรื่นแม้ในกรณีที่ระบบปิดรับการจองครับ
